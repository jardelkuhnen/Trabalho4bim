# Trabalho4bim

Menu de Usuário

1 - Cadastros
	1.1 Cadastro Cliente
		- Salvar cliente
			Para que seja possível cadastrar um cliente, é necessário que seja informado um Id para o cliente, seu 			nome, telefone, endereço, cidade e email.
			Para estado e gênero, é necessário escolher uma opção das quais ja estão auto cadastradas no software.
			Lembrando que os campos id e nome não podem ser nulos.	
			Ao inserir todas as informaçoes clique em salvar, que o cliente será salvo no banco de dados.
		
		- Editar Cliente
			Informe o id do cliente e altere as informações desejadas, ao finalizar as alterações, 			clique em editar para que seja alterado o cadastro do cliente no banco de dados.
		
		- Excluir cliente
			Informe o id do cliente e clique em deletar, assim será excluido o cliente do banco de 			dados.	
	
	1.2 Cadastro de Produto
		- Salvar Produto
			Para cadastrar um novo produto informe um id, um código de barras, descrição, custo e margem 			de lucro. 	
			Para gênero e unidade de medida, informe uma das opções que ja estão auto cadastradas no 			software.
			Lembrando que informações como id, descricao valor e margem lucro não podem ser nulas.
