# Trabalho4bim

Menu de Usuário


Informações de acesso inicial:
	Para acessar o Software pelaprimeira vez, utilize o usuario: 1 e senha: 1, que já estão préviamente gravados no banco ao ser criado e populado com os 	dados iniciais.
	


1 - Cadastros
1.1 Cadastro Cliente
- Salvar cliente
Para que seja possível cadastrar um cliente, é necessário que seja informado um Id para o cliente, seu nome, telefone, endereço, cidade e email.
Para estado e gênero, é necessário escolher uma opção das quais ja estão auto cadastradas nosoftware.
Lembrando que os campos id e nome não podem ser nulos.	
Ao inserir todas as informaçoes clique em salvar, que o cliente será salvo no banco de dados.
		
- Editar Cliente
Clique F2 do teclado para informar o nome do cliente e clique em buscar para localizar os mesmos. Ao realizar a busca dê um duplo clique do mouse para que os dados do cliente sejam replicados para a tela de cadastro para serem realizadas as devidas alterações.
Cliqe em salvar para salvar alterações.
		 
- Excluir cliente
Clique F2 para realizar abrir a tela de consulta, onde deverá ser informado o nome do cliente para consulta. Dê um duplo clique na no linha de cadastro para carregar os dados do cliente, e após isso clique em excluir.
	
1.2 Cadastro de Produto
- Salvar Produto
Para cadastrar um novo produto informe um id, um código de barras, descrição, custo e margem de lucro. 	
Para gênero e unidade de medida, informe uma das opções que ja estão auto cadastradas no software.
Lembrando que informações como id, descricao valor e margem lucro não podem ser nulas.
		
- Editar Produto
Clique F2 para consulta, informe a descrição do produto. 
Selecione o produto e de um duplo clique para carregar produto.
Realize as alterações necessária e clique em salvar.
	
- Excluir
Clique F2 para consultar produto.
Selecione o produto consultado, e de um duplo clique para carregar.
Após carregado clique em excluir para apagar.
		
1.3 Cadastro de Usuário
- Salvar
Insira o iddo Cliente, o id do usuário, o seu nome de usuário e senha;
Clique em salvar para salvar usuario.

- Editar	
Insiaro o id do usuário e altera as informações desejadas.
Clique em editar para alterar.

- Excluir
Informe o id do usuáro e clique em excluir.

1.4 Lançamentos/Vendas	
Informe o número de nota, o cliente, o produto, a quantidade de produtos e clique em adicionar.
Caso queira remover o produto, selecione o e clique em remover.
Insira o valor de pagamento.
Ao final de adicionar os produtos clique em gravar para finalizar a venda.
O software calculará o valor total e o valor de troco e informará em uma tela.
Clique em cancelar, caso queira cancelar a venda.
